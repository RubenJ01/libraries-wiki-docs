# RJDS documentation

Wiki content for **RJDS** public GitHub projects: PHP packages, **Magento modules**, and related tooling. Sources are the [GitHub wikis](https://github.com/RubenJ01?tab=repositories) on [@RubenJ01](https://github.com/RubenJ01); repos without a wiki are omitted.

Refresh locally with `python scripts/sync_wikis.py`, or rely on the GitHub Actions workflow after you push.
